# Sheing LWC React POC

## Installation

* `git clone git@github.com:Sheing/LWC-React-POC.git`
* cd LWC-React-POC
* npm install
* npm start
* visit `http://localhost:8080/`

## Deploy to Salesforce

* `git clone git@github.com:Sheing/LWC-React-POC.git`
* cd LWC-React-POC
* npm install
* create SFDX scratch org (first time)
* npm deploy
